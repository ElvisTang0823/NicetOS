# NicetOS
> A child-safe operating system based on Ubuntu LTS.

## About
NicetOS (Nice + Protect + OS) is a Linux distribution based on Ubuntu LTS, designed to provide a safer digital environment for children and teenagers.
The project aims to combine content filtering, parental controls, AI-assisted risk analysis, and privacy-focused design into a single operating system.

## Planned Features
Website whitelist and blacklist filtering
AI-assisted website risk analysis
Parental control dashboard
Child-friendly app store
Local-first privacy protection
Customized Ubuntu environment

## Development Status
Early Development
The project is currently in the planning and prototyping stage.
